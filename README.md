# Superstore Sales Analysis Project

## Project Overview
Analyzed Superstore sales, returns, and regional manager data to identify key business insights. The project demonstrates advanced SQL skills and provides actionable insights on sales trends, customer behavior, product performance, and regional profitability.

## Tools & Skills
- **SQL (PostgreSQL)**: Data extraction, joins, aggregations, subqueries, window functions, date functions.  
- **Data Analysis Skills**: Trend analysis, return rate analysis, profitability, and top-performing product/customer identification.  

## Datasets / Tables
1. **superstore_orders** – Order-level sales data with customer, product, and shipping details.  
2. **superstore_returned_orders** – Returned order records for return rate analysis.  
3. **superstore_people** – Regional managers and their assigned regions.  

## Key Analyses
- **Sales & Profit Analysis**
  - Total orders, total sales, total profit.
  - Sales by region, category, sub-category, and segment.
  - Top-selling and most profitable products.
  - Profit margins and loss-making products.

- **Customer & Returns Analysis**
  - Top customers by sales and profit.
  - Return rate calculation and most returned products/categories.
  - Regions with highest return rates.

- **Trends & Insights**
  - Monthly and yearly sales trends.
  - Average delivery time.
  - Shipping mode performance.
  - Regional manager performance and state-wise profits.

## Key Insights
- East region generated the highest total sales.  
- Furniture category contributed the most to revenue; Office Supplies had the highest order volume.  
- Some products consistently made losses, suggesting inventory optimization opportunities.  
- Certain regions and categories had higher return rates, providing actionable insights for operational improvement.  
- Customers using credit cards showed higher repeat purchase behavior.
